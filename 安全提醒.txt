密码管理：
---------

【如果账号被盗怎么办？】
================================================================================

1. 立即修改密码

2. 撤销所有Personal Access Token：
   Settings → Developer settings → Tokens
   删除所有可疑Token

3. 检查仓库是否有异常提交：
   进入仓库 → Commits
   查看是否有不是您的提交

4. 如有异常，恢复到之前的版本：
   git reset --hard <commit-hash>
   git push -f origin main

5. 联系GitHub支持
